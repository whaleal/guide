## Manager Deployment

```
Manager Deployment 可以执行以下操作：
 - Cluster Operation
```

### Cluster Operation

集群操作

a. 进入页面左侧导航栏

b. 点击 MongoDB 选项按钮，选择 MongoList 选项

c. 在 MongoDB 静态信息页面，点击操作（更新节点信息、启动集群、关闭集群、重启集群、脱离纳管、重命名）
